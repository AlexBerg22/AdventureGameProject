A simple game that alows the player to buy items, fight monsters, and move around on a map.
Sound effects and music are from public libraries, EarthBound (Nintendo, 1994), or Deltarune (Toby Fox, 2018).

This game requires the pygame and playsound3 libraries to run.
You can install them individually, or run the following command in the terminal using the provided requirements file:

pip install -r requirements.txt

Ensure you navigate to the requirmenents.txt file's location before runing the pip install command, or it won't work.
